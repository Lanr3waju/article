## Development

- Clone the project

```bash
git clone https://github.com/Lanr3waju/articles
```

- Enter Project Path

```bash
cd articles
```

- Install Dependencies

```bash
npm install
```

You can check against Prettier:

```bash
npm run format:check
```

and to have it actually fix (to the best of its ability) any format issues, run:

```bash
npm run format
```
